## AI味降低指南

**核心原则**：让文章更像人类写的

| 问题 | 解决方案 |
|------|----------|
| 句式单一 | 长短句交替，避免统一长度 |
| 模板化表达 | 去除「与此同时」「值得注意的是」 |
| 情感直说 | 通过动作/表情暗示而非直接描述 |
| 过度解释 | 相信读者理解能力，减少说明性文字 |
| 完美对称 | 打破整齐结构，增加自然节奏 |
| 缺乏细节 | 加入具体感官细节和生活化场景 |
| 连接词过多 | 减少「然后」「于是」「因此」 |

**检测清单**：
- [ ] 句式长度是否有变化？
- [ ] 是否有口语化表达？
- [ ] 情感是否通过动作暗示？
- [ ] 是否有模板化词汇？
- [ ] 细节是否具体生动？
